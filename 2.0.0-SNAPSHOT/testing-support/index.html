The Testing Support Module

Overview

The Oracle Tools Testing Support module provides a number of extensions to simplify testing concurrent and distributed applications.

As this module makes significant use of the concept of Deferred References, we recommend reading the Introduction to Deferred References for initial examples and background information.

Distribution

The Oracle Tools Testing Support module is distributed as a single jar called: oracle-tools-testing-support-2.0.0-SNAPSHOT.jar

The maven.java.net artifact information is as follows:

  • Maven Group Id: com.oracle.tools
  • Maven Artifact Id: oracle-tools-testing-support
  • Maven Version: 2.0.0-SNAPSHOT

Alternatively you can downlown directly from: oracle-tools-testing-support-2.0.0-SNAPSHOT.jar

Source Code

Source code for this and other Oracle Tools modules can be found here: https://github.com/coherence-community/oracle-tools
