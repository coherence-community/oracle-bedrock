The Runtime Module for Oracle Tools

Overview

The Oracle Tools Runtime module provides a high-level framework that allows developers to define, create, monitor and manage other applications and associated processes at runtime.

An introduction to these concepts is covered by the Introduction to Application Runtime Management.

Distribution

The Oracle Tools Runtime module is distributed as a single jar called: oracle-tools-runtime-2.0.0-RC6-SNAPSHOT.jar

The maven.java.net artifact information is as follows:

  • Maven Group Id: com.oracle.tools
  • Maven Artifact Id: oracle-tools-runtime
  • Maven Version: 2.0.0-RC6-SNAPSHOT

Alternatively you can downlown directly from: oracle-tools-runtime-2.0.0-RC6-SNAPSHOT.jar

Source Code

Source code for this and other Oracle Tools modules can be found here: https://github.com/coherence-community/oracle-tools
